1. PALETA DE COLORES

ÚNICOS colores permitidos:

Amarillo principal: #FFC502

Azul oscuro: #0F1923

Negro: #000000

Negro suave: #0F0F0F

Blanco: #FFFFFF

REGLAS

amarillo = acentos + CTA + verificado

azul oscuro = texto + bordes

negro = fondos fuertes

blanco = fondo principal

NO usar otros colores.

1. TIPOGRAFÍA

ÚNICA fuente:

Montserrat

Pesos permitidos:

Regular

Medium

SemiBold

Bold

JERARQUÍA BASE

Hero titles → Bold / ExtraBold (grande)

Subtítulos → Bold / 24px

Texto principal → Medium / 16–24px

Descripción → Medium / 11–14px

1. SISTEMA DE BORDES

Regla clave del sistema:

bordes: 0.5px

color: #0F1923

radio: suave (consistente)

PROHIBIDO

outlines negros duros

sombras tipo material design agresivas
