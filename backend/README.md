# Goberna Club Backend

Backend en Express + TypeScript + Prisma + MySQL.

## Caracteristicas

- Swagger en `/api/docs`
- Prisma sobre MySQL 8+
- bootstrap que crea la base si no existe
- sincronizacion del esquema al arrancar
- seed de usuarios demo para integrar frontend
- modulos de auth, users, profiles, consultants-public, admin, reviews, audit, notifications y health
